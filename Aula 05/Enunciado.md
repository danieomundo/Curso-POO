Criar Classe ContaBanco 

<b>Atributos:</b>

- public numConta
- protected tipo (se CC ou CP)
- private dono
- private saldo
- private status
        
<b>Métodos:</b> 

- abrirConta() // abrir CC ganha 50,00. abrir CP ganha 150,00.
- fecharConta() // não pode ter dinheiro na conta.
- depositar() 
- sacar()
- pagarMensal() // mensalidade CC: 12,00. mensalidade CP: 20,00.
